# Go pdfjam

A [pdfjam](https://github.com/rrthomas/pdfjam) web interface for joining PDF format files.

## Requirements

- go
- pdfjam

## Running

Use:

```sh
go run main.go
```

to start the http server on port 8080.

## Building

```sh
go build
```

## Running on docker

```sh
docker-compose up -d
```
